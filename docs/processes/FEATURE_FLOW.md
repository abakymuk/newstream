# Feature Flow

1. User Story + Acceptance Criteria
2. UX sketch
3. Contract (API/data schema)
4. DB migrations
5. Backend implementation
6. Tests
7. Frontend implementation
8. Feature flag
9. Deploy (preview → staged rollout)
10. Closure (docs, flag removed, KPI checked)
